    Enumeration (GI Enum)

    Gtk-4.0 ▶ Gtk ▶ AssistantPageTypeDeprecated 4.10

    Determines the role of a page inside a Gtk.Assistant.

    The role is used to handle buttons sensitivity and visibility.

    Note that an assistant needs to end its page flow with a page of type Gtk.AssistantPageType.CONFIRM, Gtk.AssistantPageType.SUMMARY or Gtk.AssistantPageType.PROGRESS to be correct.

    The Cancel button will only be shown if the page isn’t “committed”. See gtk_assistant_commit() for details.

    Deprecated

    since 4.10: Gtk.Assistant will be removed in GTK 5
