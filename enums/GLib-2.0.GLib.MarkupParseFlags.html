    IGNORE_QUALIFIED

    IGNORE_QUALIFIED: number

    Ignore (don't report) qualified attributes and tags, along with their contents. A qualified attribute or tag is one that contains ':' in its name (ie: is in another namespace). Since: 2.40.

    • Defined in glib-2.0/glib-2.0.d.ts:15028

    PREFIX_ERROR_POSITION

    PREFIX_ERROR_POSITION: number

    Normally errors caught by GMarkup itself have line/column information prefixed to them to let the caller know the location of the error. When this flag is set the location information is also prefixed to errors generated by the GLib.MarkupParser implementation functions

    • Defined in glib-2.0/glib-2.0.d.ts:15021

    TREAT_CDATA_AS_TEXT

    TREAT_CDATA_AS_TEXT: number

    When this flag is set, CDATA marked sections are not passed literally to the passthrough function of the parser. Instead, the content of the section (without the <![CDATA[ and ]]>) is passed to the text function. This flag was added in GLib 2.12
