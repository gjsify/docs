    Gio-2.0 ▶ Gio ▶ SettingsBindFlags

    Flags used when creating a binding.

    These flags determine in which direction the binding works. The default is to synchronize in both directions.

    GET_NO_CHANGES

    GET_NO_CHANGES: number

    When set in addition to Gio.SettingsBindFlags.GET, set the GObject.Object property value initially from the setting, but do not listen for changes of the setting

    • Defined in gio-2.0/gio-2.0.d.ts:6652

    INVERT_BOOLEAN

    INVERT_BOOLEAN: number

    When passed to Gio.Settings.bind, uses a pair of mapping functions that invert the boolean value when mapping between the setting and the property. The setting and property must both be booleans. You cannot pass this flag to Gio.Settings.bind_with_mapping.

    • Defined in gio-2.0/gio-2.0.d.ts:6659

    NO_SENSITIVITY

    NO_SENSITIVITY: number
