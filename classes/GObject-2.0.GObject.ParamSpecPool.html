    GObject-2.0 ▶ GObject ▶ ParamSpecPoolAbstract

    A GObject.ParamSpecPool maintains a collection of GParamSpecs which can be quickly accessed by owner and name.

    The implementation of the GObject.Object property system uses such a pool to store the GParamSpecs of the properties all object types.
