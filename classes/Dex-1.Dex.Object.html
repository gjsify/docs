    Class (GI Class)

    Dex-1 ▶ Dex ▶ ObjectAbstract

    Dex.Object is the basic building block of types defined within libdex. Futures, Schedulers, and Channels all inherit from DexObject which provides features like thread-safe weak pointers and memory management operations.

    Objects that are integrating with GIO instead inherit from their natural type in GIO.
