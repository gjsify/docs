    Class (GI Struct)

    GObject-2.0 ▶ GObject ▶ TypeInfo

    This structure is used to provide the type system with the information required to initialize and destruct (finalize) a type's class and its instances.

    The initialized structure is passed to the g_type_register_static() function (or is copied into the provided GObject.TypeInfo structure in the g_type_plugin_complete_type_info()). The type system will perform a deep copy of this structure, so its memory does not need to be persistent across invocation of g_type_register_static().
