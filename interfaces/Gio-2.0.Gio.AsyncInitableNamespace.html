    • newv_async(

          cancellable?: Gio.Cancellable | null,
          callback?: AsyncReadyCallback<Gio.AsyncInitable<GObject.Object>> | null,
      ): void

      Helper function for constructing Gio.AsyncInitable object. This is similar to g_object_newv() but also initializes the object asynchronously.

      When the initialization is finished, callback will be called. You can then call g_async_initable_new_finish() to get the new object and check for any errors.

      Parameters

      • object_type: GType

        a GObject.GType supporting Gio.AsyncInitable.

      • n_parameters: number

        the number of parameters in parameters

      • parameters: Parameter

        the parameters to use to construct the object

      • io_priority: number

        the I/O priority of the operation

      • Optionalcancellable: Gio.Cancellable | null

        optional Gio.Cancellable object, null to ignore.

      • Optionalcallback: AsyncReadyCallback<Gio.AsyncInitable<GObject.Object>> | null

        a Gio.AsyncReadyCallback to call when the initialization is finished

      Returns void
